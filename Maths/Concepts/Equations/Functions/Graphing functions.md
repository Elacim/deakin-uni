##### Stationary points
For a quadratic like $y=ax^{2}+bx+c$, the stationary point is always located at $x=\frac{-b}{2a}$

Complete the square formula:
$y=(x+\frac{b}{2})^{2} + c - \frac{b}{2}^{2}$

$ax^{2}+bx+c \ce{->} a(x+ \frac{b}{2a})^{2} + c - \frac{b^{2}}{4a}$
