#### Translation
###### Horizontal
$y = f(x+c)$
A positive number translates... :: Left ($-x$)
And a negative translates... :: Right ($+x$)
###### Vertical
$y = f(x) + c$
A positive number translates... :: Upwards ($+y$) 
And a negative translates... :: Downwards ($-y$)
