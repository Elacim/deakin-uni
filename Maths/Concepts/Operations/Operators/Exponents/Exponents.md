**Exponentiation** :: Raise the base ($b$) by exponent ($e$) to get the resulting number ($n$)

$2^{3^{2}} = 2^{9} = 512$

$log_{2}(\frac{1}{8})$
-> $a^{-n} = \frac{1}{a^{n}}$
-> $2 \times 2 \times 2 = 8$
-> $\frac{1}{2^{3}} = 2^{-3}$
	-> $2^{-3} = 2 \times \frac{1}{2} \times \frac{1}{2} \times \frac{1}{2} = \frac{1}{8}$
