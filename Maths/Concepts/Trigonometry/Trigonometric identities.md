#### Basic trigonometric (pythagorean) identity
$\cos^{2} \theta + \sin^{2} \theta = 1$
where
-> $\cos^{2} \theta = (\cos \theta)^{2}$
-> $\sin^{2} \theta = (\sin \theta)^{2}$

#### For any angle $\theta$
$\sin(-\theta) = -\sin \theta$
$\cos(-\theta) = \cos \theta$
$\tan(-\theta) = -\tan \theta$

$\cos^{2} \theta + \sin^{2} \theta = 1$
All $\sin^{2}(\theta) + \cos^{2}(\theta)$ values are $=1$
